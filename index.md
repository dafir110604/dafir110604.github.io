## Installation

To install the DelusMessenger app, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/dafir110604/DelusMessanger.git
   ```

2. Navigate to the project directory:
   ```bash
   cd DelusMessanger
   ```

3. Install dependencies:
   ```bash
   flutter pub get
   ```

4. Run the app:
   ```bash
   flutter run
   ```
